🎨 SVG Tools

A little workshop for building Learn 2 Draw drawings. Draw a picture, refine it step by step, and learn how the SVG behind it works — then hand the result to Claude.

✏️
Step 1 · create

Draw → SVG

Sketch a picture with your mouse (or trace a photo). Every stroke is captured as a smooth SVG path, grouped into steps. Hit Play to watch it draw itself, then copy the result.
🛠️
Step 2 · refine

Step Editor

Paste in a drawing's steps and walk through them one at a time. Edit any stroke's SVG live, or turn on drag points to reshape the curves by hand. Then copy it back.
🧪
Learn · how it works

SVG Playground

See how SVG actually works: a circle you control with sliders, a path you can type, and a jersey-body curve whose control points you drag to feel how curves bend.
Typical flow: ✏️ Draw a picture  →  🛠️ refine each step  →  📋 Copy for Claude and paste it back with the name, emoji, level and base colour. New to SVG? Start in the 🧪 Playground.
